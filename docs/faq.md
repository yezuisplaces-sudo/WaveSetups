# Sik Sorulan Sorular

## Plugin neden aktif olmuyor?

Cogu durumda Java surumu uyumsuz oluyor. Java 17 veya ustu kullandiginizi kontrol edin.

## Config degisikligi aninda uygulanir mi?

Bazi ayarlar icin `/reload` yeterli olur, bazilari icin tam sunucu restart gerekir.

## Lisans tek sunucu mu?

Lisans modeline gore degisir. Satin aldiginiz urunun aciklamasindaki lisans notlarini okuyun.

## Destek suresi ne kadar?

Aktif destek saatleri ve ortalama yanit suresi [Destek Kurallari](/support) sayfasinda yer alir.
