## Paket Mantigi

Paketler birden fazla pluginin birbiriyle uyumlu calisacagi sekilde gelir:

- Performans odakli varsayilanlar
- Cakisma azaltan config secimleri
- Roleplay ve ekonomi sistemine uygun izin yapisi
