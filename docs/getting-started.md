# Hizli Baslangic

WaveSetups urunlerinin tamamini ayni mantikta kurabilirsiniz.

## Gereksinimler

- Java 17+
- Paper / Purpur 1.20.x veya ustu
- En guncel plugin dosyasi (`.jar`)

## Standart Kurulum

1. Sunucunuzu kapatin.
2. Plugin dosyasini `plugins/` klasorune atin.
3. Sunucuyu baslatin ve ilk config dosyalarinin olusmasini bekleyin.
4. Sunucuyu tekrar kapatin.
5. Olusan config dosyalarini duzenleyin.
6. Sunucuyu tekrar acin.

## Guncelleme

1. Eski `.jar` dosyasini yedekleyin.
2. Yeni `.jar` dosyasiyla degistirin.
3. Changelog sayfasindaki `breaking changes` notlarini kontrol edin.
4. Gerekirse config migration adimlarini uygulayin.

## Destek Icin Gerekli Bilgiler

Destek talebi acarken su bilgileri paylasin:

- Sunucu surumu (`/version`)
- Plugin surumu
- Hata logu (tam stack trace)
- Yaptiginiz son degisiklik

Detayli destek akisi icin [Destek Kurallari](/support) sayfasina bakin.
