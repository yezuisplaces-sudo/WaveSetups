## Özellikler

- Join item sistemi
- Otomatik scoreboard
- NPC tabanlı sunucu yönlendirme
- Çift dil mesaj desteği

## Kurulum

1. Plugin dosyasını `plugins/` klasörüne atın.
2. `/ls setup` komutunu çalıştırın.
3. NPC ve item aksiyonlarını config dosyalarından ayarlayın.

## Komutlar

| Komut | Açıklama | Yetki |
| --- | --- | --- |
| `/ls setup` | İlk kurulum sihirbazını açar | Admin |
| `/ls reload` | Config dosyalarını yeniler | Admin |
| `/hub` | Lobiye döner | Oyuncu |
