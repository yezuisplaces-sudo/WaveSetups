## Uyumlu Sürümler


- Paper/Purpur 1.20.x - 1.21.x
- Java 17+


## Kurulum


1. `WaveCoreEssentials-x.x.x.jar` dosyasını `plugins/` klasörüne atın.
2. Sunucuyu başlatıp varsayılan config dosyalarını oluşturun.
3. `plugins/WaveCoreEssentials/config.yml` dosyasını düzenleyin.
4. `/wce reload` ile ayarları yenileyin.



## Komutlar
| Komut | Açıklama | Yetki |
| --- | --- | --- |
| `/sethome <isim>` | Yeni home kaydeder | Oyuncu |
| `/home <isim>` | Home noktasına ışınlar | Oyuncu |
| `/spawn` | Spawn noktasına gider | Oyuncu |
| `/wce reload` | Plugini yeniden yükler | Admin |



## Permission Node'ları



```txt
wavesetups.core.home
wavesetups.core.sethome
wavesetups.core.spawn
wavesetups.core.reload
```
